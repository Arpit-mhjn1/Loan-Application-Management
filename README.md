# Loan-Application-Management-System
Java | SpringBoot | Thymeleaf | PostgreSQL

## Project Overview

The Loan Application Management System is a Java-based application designed to streamline and automate the process of managing loan applications. The system enables users to submit loan requests, while administrators can review, approve, or reject applications efficiently.

This project demonstrates the use of Java, SQL, and database connectivity to build a structured financial application system.


---

## Tech Stack

**Programming Language:** Java

**Database:** SQL / MySQL

**Connectivity:** JDBC

**IDE:** Eclipse / IntelliJ IDEA / NetBeans

**Version Control:** GitHub



---

## Features

User registration and login system

Loan application submission

View loan application status

Admin panel to manage applications

Approval and rejection of loan requests

Secure database storage for user and loan details
